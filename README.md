# HyperHotkey - WebGL Background Automation Bot

เครื่องมือสำหรับช่วยกดปุ่มคีย์บอร์ดและเมาส์ในเบราว์เซอร์อัตโนมัติแบบทำงานในพื้นหลัง (Background Automation) เหมาะสำหรับเกม WebGL (เช่น Flyff Universe) บอทส่งปุ่มเข้าแท็บเกมได้โดยตรง แม้จะย่อหน้าจอ ย้ายหน้าต่าง หรือดู YouTube/เล่นเกมอื่นทับอยู่

---

## 🌟 1. ความสามารถหลัก (Features)

*   **Background Controlling:** ส่งปุ่มกดและคลิกเข้าแท็บเกมในพื้นหลังโดยไม่ต้องโฟกัสหน้าจอ หรือแย่งเมาส์หลัก
*   **Dual Clients (ใหม่):** รองรับการเปิด 2 เบราว์เซอร์/ไอดีพร้อมกัน แยกเซสชันอิสระออกจากกัน (`chrome-profile` และ `chrome-profile-2`)
*   **Target Client Routing (ใหม่):** กำหนดเป้าหมายของแต่ละคำสั่งได้อิสระว่าต้องการให้ส่งปุ่มไปที่เบราว์เซอร์ 1 (หลัก), เบราว์เซอร์ 2 (รอง) หรือพร้อมกันทั้งคู่
*   **Autoload Start URL (ใหม่):** ดึงคีย์เวิร์ดเว็บเกมจากโปรไฟล์ล่าสุดไปเปิดหน้าเว็บให้ทันทีเมื่อเริ่มโปรแกรม
*   **Control Panel Auto-tab (ใหม่):** เปิดหน้าตั้งค่า `localhost:3000` เป็นแท็บที่ 2 ในหน้าต่างเบราว์เซอร์แรกโดยอัตโนมัติ (ไม่เปิดหน้าต่างป๊อปอัปแยก) และทำความสะอาดหน้าแท็บว่างที่ค้างอยู่ให้เรียบร้อย
*   **Reliable Native Hook (ใหม่):** ดีเลย์การเปิดใช้งานระบบดักจับเมาส์และคีย์บอร์ดระดับ OS จนกว่าเบราว์เซอร์จะพร้อม ป้องกันปัญหาปุ่มค้าง/สะดุดช่วงเริ่มเปิดบอท
*   **Global Hotkey Capture:** ดักฟังคีย์บอร์ด (`F1-F12`, `INSERT` ฯลฯ) และปุ่มด้านข้างเมาส์ (Button 4/5) ได้จากทุกหน้าต่างโปรแกรม
*   **Human-Like Jitter:** ระบบสุ่มดีเลย์กดลง (10-35ms) ค้างปุ่ม (60-130ms) และดีเลย์รอบ (Jitter) ป้องกันการโดนแบน

## 📦 2. การติดตั้ง (Installation)

1.  ติดตั้ง **Node.js (v18 ขึ้นไป)** จาก [nodejs.org](https://nodejs.org/)
2.  ดับเบิลคลิก `1 install.bat` เพื่อติดตั้ง Library
3.  ดับเบิลคลิก `2 playwright install.bat` เพื่อติดตั้งเบราว์เซอร์จำลอง

---

## 🎮 3. วิธีใช้งาน (Usage Guide)

1.  **เริ่มรันบอท:** ดับเบิลคลิก `3 start.bat` หรือรันคำสั่ง `npm start`
2.  **ตั้งค่าเริ่มต้นในคอนโซล:**
    *   เลือกโหมดการทำงาน: `1` สำหรับ Single Client (1 จอ) หรือ `2` สำหรับ Dual Clients (2 จอ)
    *   เลือกประเภทเบราว์เซอร์ที่ต้องการใช้งาน (Google Chrome, Microsoft Edge, Mozilla Firefox)
3.  **เข้าเล่นเกม:**
    *   บอทจะเปิดหน้าต่างเกมและเปิดหน้า Control Panel ในหน้าต่างที่ 1 ให้อัตโนมัติ (และจอ 2 หากเลือก Dual)
    *   เมื่อหน้าเว็บเกมโหลดสมบูรณ์ บอทจะล็อกเป้าหมายให้เองโดยแสดง `✅ Game tab detected!`
4.  **ตั้งค่าปุ่มลัดและลำดับวนซ้ำ:**
    *   เปิดหน้าเว็บตั้งค่า: **[http://localhost:3000](http://localhost:3000)** บนเบราว์เซอร์ของคุณ
    *   จัดการโปรไฟล์และคำสั่งการรันได้ 3 รูปแบบ:
        *   **Loop:** วนซ้ำตามเวลาที่กำหนด (เช่น ปุ่มช่วยกดฮีล)
        *   **Buff Sequence:** สคริปต์กดชุดปุ่มเรียงกันแบบตั้งเวลาดีเลย์ (เช่น กดร่ายบัฟ 1-7)
        *   **Single Press:** กดปุ่มส่งเข้าไปครั้งเดียวเมื่อกดปุ่มลัด
    *   กำหนดตัวเลือก **Target Client** เพื่อระบุปลายทางส่งสัญญาณปุ่มกด

## 🕹️ ปุ่มลัดเริ่มต้น (Default Hotkeys)

*(ปรับแต่งปุ่มและเวลาได้เองผ่านหน้าเว็บตั้งค่า ทุกการบันทึกจะซิงค์การทำงานแบบเรียลไทม์โดยไม่ต้องปิดเปิดโปรแกรมใหม่)*

*   **`F9` หรือ ปุ่มข้างเมาส์ Back (Button 4):** เปิด/ปิด การวนลูปฮีลหลัก (Healing Loop)
*   **`F10` หรือ ปุ่มข้างเมาส์ Forward (Button 5):** เปิด/ปิด การวนลูปฮีลย่อย (Mini Healing Loop)
*   **`INSERT`:** รันชุดลำดับบัฟ (Buff Sequence)
