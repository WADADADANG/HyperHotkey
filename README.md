# HyperHotkey - WebGL Background Automation Bot

เครื่องมือสำหรับช่วยกดปุ่มคีย์บอร์ดและเมาส์ในเบราว์เซอร์อัตโนมัติแบบทำงานในพื้นหลัง (Background Automation) เหมาะสำหรับการเล่นเกม WebGL (เช่น Flyff Universe) โดยบอทจะทำงานส่งปุ่มควบคุมเข้าไปในแท็บเกมได้โดยตรง แม้ว่าคุณจะย่อหน้าจอเบราว์เซอร์ลง หรือกำลังทำงาน/ดู YouTube/เล่นเกมอื่นทับอยู่ก็ตาม!

---

## 🌟 1. ความสามารถของโปรแกรม (Features)

*   **Background Controlling (ส่งปุ่มเข้าเบราว์เซอร์ในพื้นหลัง):** ส่งปุ่มกดและจำลองการคลิกเข้าสู่แท็บเกมได้โดยตรงโดยไม่ต้องนำหน้าจอเบราว์เซอร์ขึ้นมาโฟกัส หรือแย่งเมาส์การใช้งานของคุณ
*   **Global Hotkey Capture (ตรวจจับปุ่มลัดระดับระบบปฏิบัติการ):** ดักฟังปุ่มลัดจากคีย์บอร์ด (เช่น `F8`, `F9`, `F10`, `INSERT`, `HOME`) และปุ่มเมาส์เพิ่มเติม (เช่น ปุ่ม Forward/Backward ด้านข้างเมาส์) ได้จากทุกที่ แม้ในตอนที่คุณกำลังเปิดโปรแกรมอื่นอยู่
*   **Human-Like Jitter & Delay (ระบบสุ่มเวลาเพื่อความปลอดภัย):** 
    *   สุ่มดีเลย์ก่อนกดปุ่ม (10 - 35ms)
    *   สุ่มเวลาการกดปุ่มค้าง (60 - 130ms) เพื่อความเนียนเสมือนคนกดจริง
    *   สุ่มช่วงเวลา Interval (Jitter) ระหว่างรอบ ช่วยป้องกันการตรวจจับจากระบบ Anti-Cheat
*   **Persistent Context (โปรไฟล์เบราว์เซอร์แยกเฉพาะ):** ล็อกอินไอดีเกมทิ้งไว้ได้โดยไม่ต้องเข้าระบบใหม่ทุกครั้ง โดยแยกโปรไฟล์เฉพาะของโปรเจกต์ (`chrome-profile`, `edge-profile`, `firefox-profile`)
*   **Web Control Panel GUI (หน้าจอตั้งค่าใช้งานง่ายผ่านเว็บ):** ทำงานร่วมกับเว็บคอนโทรลพาเนลที่ให้คุณเลือกสร้าง แก้ไข และบันทึกโปรไฟล์การกด รวมถึงกำหนดรูปแบบ Action แบบวนลูป (Loop), ลำดับการกดบัฟ (Buff Sequence), และแบบกดครั้งเดียว (Single Press) ได้แบบเรียลไทม์

## 📦 2. สิ่งที่ต้องติดตั้ง & วิธีติดตั้งโดยละเอียด (Prerequisites & Installation)

### 📌 สิ่งที่ต้องเตรียมในเครื่องคอมพิวเตอร์
1.  **Node.js (เวอร์ชัน 18 ขึ้นไป แนะนำ LTS):** เนื่องจากโปรแกรมเขียนด้วย Node.js 
    *   ดาวน์โหลดได้ที่ [Node.js Official Website](https://nodejs.org/)
    *   (กรุณาเลือกดาวน์โหลดแบบติดตั้งมาตรฐานสำหรับ Windows และตรวจดูให้แน่ใจว่าติดตั้งเครื่องมือเพิ่มเติมอย่าง Python/C++ Build tools ถ้าตัวติดตั้ง Node.js ถามถึง เพื่อใช้คอมไพล์ Native Node Modules)

---

### 💻 ขั้นตอนการติดตั้งโปรเจกต์
คุณสามารถติดตั้งอย่างง่ายดายด้วยสคริปต์ไฟล์ `.bat` ที่เตรียมไว้ให้ในโฟลเดอร์:

1.  **ขั้นตอนที่ 1 - ติดตั้ง Library ของ Node.js:**
    ดับเบิลคลิกเปิดไฟล์ `1 install.bat` (จะรันคำสั่ง `npm install` เพื่อลงโมดูลที่เกี่ยวข้องทั้งหมด เช่น Playwright, Global-Key-Listener)
2.  **ขั้นตอนที่ 2 - ติดตั้งเบราว์เซอร์จำลอง (Playwright Browsers):**
    ดับเบิลคลิกเปิดไฟล์ `2 playwright install.bat` (จะรันคำสั่ง `npx playwright install` เพื่อโหลดระบบเบราว์เซอร์หลักที่จำเป็นสำหรับการเชื่อมต่อแท็บเกม)

## 🎮 3. วิธีการใช้งานอย่างละเอียด (Usage Guide)

### 🚀 ขั้นตอนการเปิดใช้งานบอท
1.  **เปิดบอท:** ดับเบิลคลิกไฟล์ `3 start.bat` หรือเปิด Terminal/PowerShell ในโฟลเดอร์นี้แล้วพิมพ์ `npm start`
2.  **เลือกเบราว์เซอร์:** หน้าต่าง Command Prompt จะขึ้นให้คุณเลือกรันเบราว์เซอร์:
    ```
    =================================================
    Please select browser to run the game:
     [1] Google Chrome
     [2] Microsoft Edge
     [3] Mozilla Firefox
    =================================================
    ```
    ให้พิมพ์ตัวเลข `1`, `2` หรือ `3` แล้วกด **Enter**
3.  **เปิดหน้าเกมและล็อกอิน:** 
    *   หน้าต่างเบราว์เซอร์ที่เลือกจะถูกเปิดขึ้นมาโดยอัตโนมัติ
    *   ให้คุณเข้าไปที่หน้าเว็บไซต์เกม (เช่น [Flyff Universe](https://universe.flyff.com/)) จากนั้นทำการล็อกอินและเลือกตัวละครให้พร้อมเล่น
    *   บอทจะสแกนหาคำว่า `universe.flyff.com` ใน URL เมื่อตรวจพบแล้ว บอทจะทำการล็อกเป้าหมาย (Lock Target) ทันที และจะขึ้นข้อความในหน้าต่าง Command Prompt ว่า `✅ Game tab detected!`

---

### ⚙️ การตั้งค่าปุ่มลัดและโปรไฟล์ผ่าน Web Control Panel
เมื่อเปิดใช้งานบอทแล้ว ระบบจะสร้าง Web Control Panel ขึ้นมาคู่ขนานที่พอร์ต `3000`:
1.  เปิดเว็บเบราว์เซอร์เครื่องหลักของคุณ (หรือแท็บใหม่ในเครื่องเดียวกัน) แล้วไปที่: **[http://localhost:3000](http://localhost:3000)**
2.  **จัดการโปรไฟล์ (Profiles):** คุณสามารถสร้างโปรไฟล์แยกสำหรับแต่ละตัวละคร หรือแต่ละสายอาชีพได้
3.  **ตั้งค่า Actions:**
    *   **Loop:** สำหรับการกดปุ่มซ้ำๆ ตามเวลาที่กำหนด (เช่น ปุ่มกดฮีลหลัก/รอง)
    *   **Buff Sequence:** สคริปต์กดกลุ่มปุ่มเรียงกันแบบตั้งเวลาดีเลย์ (เช่น กดร่ายบัฟเรียงกันตั้งแต่ปุ่ม 1 ถึง 7) เมื่อกดใช้งานจะหยุดการทำงานของ Loop ตัวอื่นชั่วคราวและกลับมาทำ Loop ต่อเมื่อกดบัฟเสร็จสิ้น
    *   **Single Press:** กดปุ่มส่งเข้าไปครั้งเดียวเมื่อกดปุ่มลัด
4.  เมื่อทำการเพิ่ม/แก้ไขข้อมูลในหน้านี้ ระบบจะบันทึกลงในไฟล์ `config.json` ให้โดยอัตโนมัติแบบเรียลไทม์โดยบอทไม่ต้องรีสตาร์ทใหม่

---

### 🕹️ ปุ่มลัดเริ่มต้น (Default Hotkeys)
*(คุณสามารถเข้าไปปรับเปลี่ยนปุ่มเหล่านี้ได้ตามใจชอบที่หน้าเว็บตั้งค่า)*

*   **`F9` หรือ ปุ่มเมาส์ Back (XBUTTON1):** เปิด/ปิด การวนลูปฮีลหลัก (Healing Loop)
*   **`F10` หรือ ปุ่มเมาส์ Forward (XBUTTON2):** เปิด/ปิด การวนลูปฮีลย่อย (Mini Healing Loop)
*   **`INSERT`:** สั่งให้กดคีย์ลัดเรียงคีย์บัฟตามที่ตั้งค่าไว้ (Buff Sequence)
